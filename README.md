# 🌙 Eid Mubarak Celebration Website

A beautiful, interactive, and fully animated website to celebrate the joyous occasion of Eid. Built with modern web technologies to provide a premium and immersive experience.

## ✨ Features

- **Smooth Animations**: Powered by Framer Motion for a premium feel.
- **Interactive Wishes**: Generate personalized Eid wishes with a realistic typing effect and confetti celebration.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop screens.
- **Modern UI**: Clean, elegant design with a focus on ease of use and aesthetics.
- **Live Characters**: Whimsical animated characters to bring the site to life.

## 🛠️ Tech Stack

- **React 19**: modern UI library.
- **Vite**: Ultra-fast build tool.
- **Tailwind CSS**: Utility-first CSS framework for custom styling.
- **Framer Motion**: Industry-standard animation library.
- **Canvas Confetti**: For celebratory effects.

## 🚀 Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/hanzlashahzad01/Eid-Mubarak-Website.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
4. Build for production:
   ```bash
   npm run build
   ```
